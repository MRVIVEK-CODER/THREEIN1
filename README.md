#### Install script on Termux
```bash
$ pkg update && pkg upgrade
$ pkg install python2
$ pkg install git
$ git clone https://github.com/MRVIVEK-CODER/THREEIN1
$ pip2 install requests bs4
$ pip2 install futures
$ pkg install nodejs-lts
```
#### Run script
```bash
$ cd THREEIN1
$ python2 YUTIN.py
```
